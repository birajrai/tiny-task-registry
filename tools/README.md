# 🛠 Tools & Infrastructure

This directory contains the core validation engines, linters, and automated quality assurance scripts for the Tiny-Task Registry.

## 🗂 Key Scripts

- [**`validate-registry.js`**](scripts/validate-registry.js): Schema consistency and integrity engine.
- [**`ai-reviewer.js`**](scripts/ai-reviewer.js): AI-powered quality, security, and structure analyzer.
- [**`generate-catalog.js`**](scripts/generate-catalog.js): Automated indexer for registry discovery.

## ⚙️ Usage

To run the full validation suite locally, use `npm run validate`.
