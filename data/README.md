# 📂 Registry Data

This directory contains the automated metadata indices, search catalogs, and high-speed discovery databases for the Tiny-Task Registry.

## 🗂 Key Files

- [**`catalog.json`**](catalog.json): The primary auto-generated index of all skills in the registry.

## ⚙️ Maintenance

This directory is primarily maintained by the [Catalog Generator](../tools/scripts/generate-catalog.js) script. Do not manually edit the files in this directory unless necessary.
