# WordPress SEO Blogwriter Skill ✍️🚀

A professional, senior-level content strategy skill designed to produce high-intent, SEO-optimized blog posts ready for direct publication in the WordPress block editor.

## 🌟 Key Features

- **Truth Boxes**: Immediate value for readers with a 5-insight summary table at the start.
- **Fact-First Integrity**: Built-in mechanisms to avoid hallucinations and maintain authority.
- **WordPress Native**: Optimized formatting to ensure a perfect "copy-paste" experience into the Gutenberg editor.
- **Complete SEO Package**: Generates meta titles, descriptions, slugs, and social metadata.
- **Interactive Elements**: Includes FAQ schema (JSON-LD) and "Myth vs. Reality" sections.

## 📖 How to Use

1. **Trigger**: Activate this skill when you need a long-form article or blog post.
2. **Input**: Provide the **Title**, **Primary Keyword**, and **User Intent** (Informational, Commercial, or Transactional).
3. **Refine**: The skill will guide you through industry definitions and metadata generation before drafting the final content.
