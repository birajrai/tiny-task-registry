# WordPress Forensic UI & SEO Audit 🔍

Pixel-perfect validation for WordPress themes against original design sources.

## 🌟 Key Features

- **Side-by-Side Audit**: Systematic comparison between source (React/HTML) and target (WordPress).
- **SEO Parity**: Ensures heading structures and metadata are identical.
- **Pixel-Perfect Logic**: Strict rules for layout, spacing, and CSS class integrity.

## 📖 How to Use

1. **Trigger**: Use when finalizing a WordPress theme build or audit.
2. **Analysis**: Provide the source code and the WP template; the skill will generate a Forensic Comparison Table.
3. **Classification**: Each fix is categorized by risk level (SAFE, RISKY, BLOCKED).
