# Codebase to WordPress Converter Skill 🏗️🔄

High-fidelity conversion of static codebases (React, HTML, Next.js) into professional, production-ready WordPress themes.

## 🌟 Key Features

- **Absolute UI Lock**: Ensures 100% pixel-perfect match by preserving DOM structure and CSS classes.
- **Phased Forensic Audit**: Systematic comparison (Source vs. WP) to detect and fix UI/SEO discrepancies.
- **Technical SEO Preservation**: Guarantees meta tags, schema, and heading hierarchies remain identical to the source.
- **CMS-Driven Architecture**: Expertly maps static components to dynamic WordPress functions and ACF fields.

## 📖 How to Use

1. **Trigger**: Activate when converting a frontend project (Vite, Next.js, HTML) into a WordPress theme.
2. **Phase 1 Audit**: Provide the source code and the WordPress template; the skill will generate a UI Comparison Table.
3. **Iterative Conversion**: Follow the 4-phase forensic process to migrate logic while locking the UI.
