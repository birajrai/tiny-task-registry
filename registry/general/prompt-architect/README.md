# The Prompt Architect 🏗️✨

A high-performance meta-skill designed to assist contributors in drafting, refining, and validating new "Tiny-Tasks" for the registry.

## 🌟 Key Features

- **Standardization**: Ensures every skill follows the [Standardized Skill Anatomy](../../../docs/SKILL_TEMPLATE.md).
- **High-Intent Logic**: Optimizes instructions for clinical, unambiguous, and actionable AI execution.
- **Few-Shot Optimization**: Guides users in creating high-quality examples for better AI performance.
- **Validation-Ready**: Built-in awareness of the [Schema](../../../registry/schema.json) to ensure immediate compatibility.

## 📖 How to Use

1. **Activate**: Use this skill when you're starting a new contribution or refining an existing one.
2. **Draft**: Provide your skill's intent, and the Architect will generate the YAML frontmatter and instruction sets.
3. **Refine**: The skill will help you prune ambiguous language and add essential guardrails.
