# 📁 Registry Directory

This directory contains the core "Source of Truth" for all micro-contributions and AI-powered "Tiny-Tasks" within the ecosystem.

## 🗂 Directory Structure

- [**`general/`**](general/): Meta-skills, platform-wide instructions, and onboarding tools.
- [**`seo/`**](seo/): High-intent search engine optimization and content marketing tools.
- [**`coding/`**](coding/): Logic-driven scripts, automated code review, and development aids.

## 📏 Quality Standards

Every skill in this registry must adhere to the [Tiny-Task Schema](schema.json) and pass the automated [AI Review](../tools/scripts/ai-reviewer.js) before being merged.

For more details on how to contribute, see [CONTRIBUTING.md](../../CONTRIBUTING.md).
